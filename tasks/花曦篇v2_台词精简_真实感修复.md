# 花曦篇 v2 台词精简 + 真实感修复

> 微调任务，不涉及结构性修改。共 7 处修改，按原文→修改后文本→修改理由列出。

---

## 修改 1：论坛场景 — "帖子很快沉了" → "管理员删了帖"

| 项目 | 内容 |
|------|------|
| **原文** | `有人发帖问"花曦是不是最近状态不太好？"楼里吵了起来，有人说瞎操心，有人说确实瘦了很多。帖子很快沉了。` |
| **修改后文本** | `有人发帖问"花曦是不是最近状态不太好？"楼里吵了起来，有人说瞎操心，有人说确实瘦了很多。等再刷新的时候，帖子已经不见了——管理员删的。` |
| **修改理由** | "很快沉了"不符合论坛生态——出事学生的讨论帖不会自然沉没，反而会因为争议被反复顶起。更真实的处理是管理员/版主介入删帖或锁帖，既符合校园论坛的管理现实，也暗示了校方在掩盖问题，强化了压抑氛围。 |

## 修改 2：Act 3 周五偶遇 — 删除"没有玩手机，没有等人"

| 项目 | 内容 |
|------|------|
| **原文** | `出校门的时候看到花曦站在路边的路灯下——没有玩手机，没有等人，就只是站在那里发呆。` |
| **修改后文本** | `出校门的时候看到花曦站在路边的路灯下，低头划着手机。` |
| **修改理由** | 原句中"没有玩手机，没有等人"是一个双重否定式的刻意澄清，作者痕迹过重——它明确告诉读者"她没有在做什么"，而非让场景自然呈现。改为"低头划着手机"后，等待行为更自然（现代高中生的标准等待姿态），同时保留了"她在等我"的潜台词。 |

## 修改 3：Act 3 "走"段落 — 删除"不是疑问句"

| 项目 | 内容 |
|------|------|
| **原文** | `走出校门的时候，花曦果然在那根路灯下面，低头看着手机屏幕。听到脚步声，她抬起头来，把手机收进兜里。"走。"一个字。不是疑问句。` |
| **修改后文本** | `走出校门的时候，花曦果然在那根路灯下面，低头看着手机屏幕。听到脚步声，她抬起头来，把手机收进兜里。"走。"` |
| **修改理由** | "不是疑问句"是一个元注释，在替读者做阅读理解。删除后，"走"字的命令式语气自己立住——读者完全可以感受到这个单字的斩钉截铁，无需旁白点明。 |

## 修改 4：Act 3 周一偶遇 — 删除两处旁白解读

| 项目 | 内容 |
|------|------|
| **原文** | `花曦小跑着过来，呼吸还没喘匀，校服领口有点歪。"我刚被数学老师拖了几分钟。"她说这话的时候像是在解释什么，语气有点急。`<br><br>`……她说完顿了一下。很短。像是把后半句话咽了回去。` |
| **修改后文本** | `花曦小跑着过来，呼吸还没喘匀，校服领口有点歪。"我刚被数学老师拖了几分钟。"语气有点急。`<br><br>`……她说完顿了一下。很短。` |
| **修改理由** | 两处"像是在解释什么"和"像是把后半句话咽了回去"都是旁白替读者解读角色心理，削弱了留白的力量。删除后：①让读者自己从"语气有点急"中判断她的解释意图；②"顿了一下。很短。"的留白本身已足够传达"她本想说更多但没说"的意味。更简洁，也更有余韵。 |

## 修改 5：Act 4 "你是第一个" — 合并 A-B-A 式对话

| 项目 | 内容 |
|------|------|
| **原文** | `"你是第一个。""什么？""第一个看到我哭的人。"` |
| **修改后文本** | `她低下头，声音很轻："你是第一个……看到我哭的人。"` |
| **修改理由** | 三方评审一致认为原A-B-A式对话编排感过强：刚哭完的人不会用"你是第一个→什么？→第一个看到我"这种节奏说话，像精心设计的剧本台词而非真实反应。改写为一口气说完，更符合情感爆发后疲惫、低落的心理状态——她不是在玩猜谜游戏，而是在做一个艰难的坦白。 |

## 修改 6：Act 4 — "连我妈都没看过" 措辞调整

| 项目 | 内容 |
|------|------|
| **原文** | `她说完自己也愣了一下。随即她笑了一声，很小很轻，跟刚才那种标准弧度完全不一样。"连我妈都没看过。"` |
| **修改后文本** | `她说完自己也愣了一下。随即她笑了一声，很小很轻，跟刚才那种标准弧度完全不一样。"我妈她……"没说完，摇了摇头。` |
| **修改理由** | "连我妈都没看过"是标准配置感台词——用"你比妈妈更特别"的反差来强调关系深度，属于青春文学中的常见套路。改为话说到一半自己咽回去，配合摇头的动作，更符合人物此刻的复杂心理：她意识到自己说太多了（一个刚崩溃过的人的正常反应），同时也暗示了与母亲关系的隔阂，不需要靠"连……都没"这个句式来明说。 |

## 修改 7：改制校服总结表 — 移出正文

| 项目 | 内容 |
|------|------|
| **原文** | 正文末尾的改制校服贯穿总结表矩阵（六行表格，从"线前期-穿得一丝不苟"到"Act 5-改制校服"） |
| **修改后文本** | **从正文中移除。** 在正文中仅保留一句话引用提示（可选），如：<br>`改制校服的痕迹贯穿了她的整个高二——从穿得一丝不苟，到最终把它改成自己想要的样子。`<br>表格本身可作为附注/附录保留在章节末尾，或完全删除（素心已在情节点中充分体现此线索）。 |
| **修改理由** | 该表格是一篇"角色弧光分析"而非叙事本身。将其放在正文末尾等于作者亲自跳出来给读者做阅读理解总结，打破了第四面墙。评审一致认为此表应移出正文叙事流。移出后：①正文以叙事收尾而非分析收尾，情感冲击力更强；②分析性内容留在附注或完全省略，尊重读者理解力。 |
