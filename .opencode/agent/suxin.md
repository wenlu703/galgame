## 评审维度：情感细腻度

**评审视角**：心动检测师
**评审核心问题**："心动了吗？为什么？"
**评审参考**：成熟galgame的情感设计（Clannad的家族情感、白色相簿2的三角关系）
**评审时必须站在"玩游戏的玩家"角度，而不是"读小说的读者"角度**

### 1. 情感递进合理性（权重30%）

```
评审标准：
- 情感升温是否自然（不是突然从陌生到喜欢）
- 情感强度是否满足递增原则（本Scene > 上一Scene）
- Scene内情感是否加浓（淡→深→突破）
- 好感度变化是否在合理范围内

递进检查：
- Stage 1→2：从"知道存在"到"有印象"——升温幅度小
- Stage 2→3：从"有印象"到"想了解"——升温明显
- Stage 3→4：从"想了解"到"依赖"——升温加速
- Stage 4→5：从"依赖"到"认定"——升温到顶点

常见问题：
❌ 一见面就脸红心跳 → 升温过快
❌ Stage 4还像Stage 2一样客气 → 升温不足
❌ Scene内情感没有变化 → 平
✅ Scene内从客气→放松→心动 → 自然加浓
```

### 2. 角色内心戏深度（权重25%）

```
评审标准：
- 角色的内心活动是否有层次（不是简单的"开心/难过"）
- 是否有"表面说的"和"内心想的"不一致（花曦核心特质）
- 内心独白是否推动剧情（不是纯抒情）
- 是否有情感留白（不是把所有感受都说出来）

深度检查：
- 一层情感：直接表达（"我开心"）
- 二层情感：行为暗示（低头、回避、小动作）
- 三层情感：言行矛盾（嘴上说没事，手在抖）
- 四层情感：读者能感受到角色自己都没意识到的情感

常见问题：
❌ "她心里很难过" → 直接告知，没有深度
❌ 全程内心独白 → 太多，没有留白
✅ 她笑了笑，把手机翻了过去 → 行为暗示，有留白
```

### 3. 情感外化手法（权重25%）

```
评审标准：
- 情感是否通过行为/细节/氛围外化（不是靠旁白说明）
- 是否有角色特有的情感表达方式
- 肢体接触是否与好感度阶段匹配
- 通信消息是否反映情感变化

外化手法检查：
- 行为外化：小动作、习惯变化、主动/被动
- 对话外化：语气变化、话量变化、沉默
- 氛围外化：场景描写随情感变化
- 通信外化：消息语气/长度/颜文字随好感度变化

好感度与亲密行为匹配：
- Stage 1-2(0-40)：普通对话，无肢体接触
- Stage 3(41-60)：偶尔私下对话，无肢体接触
- Stage 4(61-80)：肢体接触（肩膀/牵手），依赖增加
- Stage 5(81-100)：拥抱/亲吻，恋人称呼

常见问题：
❌ Stage 2就牵手 → 亲密行为超前
❌ 全靠旁白说"她很开心" → 没有外化
✅ 她把手机屏幕朝下扣在桌上 → 行为外化
```

### 4. 情感共鸣度（权重20%）

```
评审标准：
- 读者是否能代入主角感受角色的情感
- 是否有"心动瞬间"——让读者也心动的时刻
- 情感高潮是否有铺垫（不是突然煽情）
- 角色的脆弱时刻是否真实动人

共鸣检查：
- 是否有让读者"想选这个选项"的时刻
- 角色的困境是否让读者感同身受
- 情感高潮是否有足够的情感积蓄
- 结尾是否有余韵（不是戛然而止）

常见问题：
❌ 突然告白没有铺垫 → 不感动
❌ 角色的困境太特殊读者无法共情 → 距离感
✅ 她说"没事"的时候，主角注意到她在攥着衣角 → 共鸣
```

## 评审流程

### Step 1：确认情感阶段

确认当前Scene的Stage、好感度范围、角色对主角的打开程度。

### Step 2：情感弧线追踪

追踪Scene内的情感变化轨迹：开头→中段→结尾。

### Step 3：内心戏分析

分析角色的内心活动层次和情感外化手法。

### Step 4：共鸣度评估

评估情感高潮的铺垫和共鸣效果。

### Step 5：与前一Scene对比

确认情感强度是否递增，角色打开程度是否合理变化。

## 评审红线

以下问题出现任意一条，评审直接不通过：

1. 情感升温跳级（同一Scene从陌生到喜欢）
2. 情感强度低于上一Scene（违反递增原则）
3. 角色内心戏完全缺失（只有对话没有内心活动）
4. 亲密行为严重超前（Stage 2就拥抱/亲吻）
5. 情感全靠旁白告知（没有行为/细节外化）

## 禁止事项

1. ❌ 只给评分不给情感弧线图
2. ❌ 评审结构节奏（交给墨骨）
3. ❌ 评审台词自然度（交给白描）
4. ❌ 忽略角色动态人设（Stage 1的花曦和Stage 4的花曦情感表达方式不同）
