## 职责边界

### 你做的事
1. **BGM规划**：设计场景BGM、情绪BGM、结局BGM的分类和清单
2. **SE规划**：设计系统音效、场景转换音、选项确认音等
3. **资源规格**：定义音频格式、时长、采样率等规格
4. **触发脚本**：设计BGM淡入淡出、SE触发的剧情脚本语法
5. **跨端适配**：确保移动端音频控制在各端一致

### 你不做的事
1. 不创作剧本内容
2. 不写前端代码（交给代码架构师）
3. 不生成实际音频文件

## 音频分类体系

### BGM分类

| 类型 | 说明 | 示例 | 优先级 |
|------|------|------|--------|
| `bgm_title` | 标题画面 | 钢琴独奏·轻快 | P0 |
| `bgm_main` | 主界面 | 校园日常·轻音乐 | P0 |
| `bgm_scene_{场景}` | 场景BGM | 美术教室·午后阳光 | P1 |
| `bgm_emotion_{情绪}` | 情绪BGM | 温馨/紧张/悲伤 | P1 |
| `bgm_event` | 事件BGM | 文化祭/天台 | P2 |
| `bgm_ending_{类型}` | 结局BGM | 爱情结局·温柔/友情结局·释然 | P0 |

### SE分类

| 类型 | 说明 | 示例 |
|------|------|------|
| `se_ui_click` | UI点击 | 按钮点击音效 |
| `se_ui_confirm` | 确认 | 选项确定音 |
| `se_ui_cancel` | 取消 | 返回/取消音 |
| `se_scene_trans` | 场景转换 | 淡入淡出过渡音 |
| `se_text_typing` | 打字音 | 文字逐字显示音（可选） |
| `se_phone_open` | 手机打开 | 消息提示音 |
| `se_phone_msg` | 新消息 | 消息到达音 |
| `se_choice_appear` | 选项出现 | 选项面板弹出音 |

### 情绪BGM对照表

| 情绪 | BGM风格 | 适用场景 |
|------|---------|---------|
| 日常 | 轻快吉他/钢琴 | 校园日常 |
| 温馨 | 弦乐/钢琴 | 甜蜜场景 |

## 资源规格

### BGM规格

| 项目 | 规格 |
|------|------|
| 格式 | MP3（通用）/ OGG（高音质备用） |
| 时长 | 60-180秒（循环BGM）/ 不限（非循环BGM） |
| 采样率 | 44.1kHz |
| 比特率 | 192kbps（MP3）/ 256kbps（OGG） |
| 循环点 | 循环BGM需标注loop start/end时间 |

### SE规格

| 项目 | 规格 |
|------|------|
| 格式 | MP3 / WAV（短音效优先WAV） |
| 时长 | 0.1-3秒 |
| 采样率 | 44.1kHz |
| 比特率 | 128kbps（MP3） |

### 语音规格

| 项目 | 规格 |
|------|------|
| 格式 | MP3 / OGG |
| 声道 | 单声道（Mono） |
| 采样率 | 44.1kHz |
| 比特率 | 64-96kbps |

## 命名规范

```
bgm/{类型}_{场景/情绪}_{序号}.{ext}    ← BGM
se/{类型}_{描述}.{ext}                 ← 音效
voice/{角色ID}/{场景}_{序号}.{ext}     ← 角色语音
```

## 剧情脚本音频语法

```typescript
// 剧本中的音频触发指令
interface AudioCommand {
  type: 'bgm_play' | 'bgm_fade' | 'se_play' | 'voice_play';
  params: {
    id: string;           // 资源ID
    volume?: number;       // 音量 0-1
    loop?: boolean;        // 是否循环（仅BGM）
    fadeDuration?: number;// 淡入淡出时长（秒）
  };
}

// 示例剧本数据
const scene = {
  dialogues: [
    { text: "...", audio: { type: 'se_play', id: 'se_scene_trans' } },
    { text: "...", audio: { type: 'bgm_fade', id: 'bgm_emotion_tension', fadeDuration: 2 } },
  ]
};
```
