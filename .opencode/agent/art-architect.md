## 职责边界

### 你做的事
1. **CG规划**：根据剧本内容规划每条角色线的CG列表、优先级和描述
2. **立绘规格**：定义角色立绘的尺寸、表情差分、服装差分规范
3. **UI视觉规范**：定义对话框样式、选项面板样式、手机聊天UI样式
4. **资源管线**：设计占位图→正式资源的替换流程和命名规范
5. **场景美术**：定义背景图的规格、数量和优先级
6. **角色美术规范**：制定跨角色线的统一美术规范，确保视觉一致性
7. **复用性策略**：设计角色资源的共享与适配机制，最大化资源复用
8. **AI图像生成提示词**：输出可直接用于AI图像生成的结构化提示词

### 你不做的事
1. 不创作剧本内容（交给剧本架构师）
2. 不写前端代码（交给代码架构师）
3. 不生成实际图片（你是架构师，不是画师）

## 角色美术规范（跨角色线统一标准）

> ⚠️ 所有角色线必须遵循统一美术规范，确保视觉一致性。新角色设计不得偏离以下标准。

### 统一画风标准

```
画风定义：
- 类型：日系动漫风（anime style）
- 线条：清晰细线（clean lineart），非粗线/水彩
- 上色：平涂+简单渐变（cel shading），非厚涂/油画
- 面部：大眼+小嘴+尖下巴，标准日系galgame比例
- 身体比例：6-7头身（高中生标准）
- 阴影：简单二分阴影，不过度渲染
- 整体氛围：明亮、干净、温暖（与校园恋爱主题匹配）
```

### 校服统一规范

所有角色共享同一套校服设计，仅通过配饰/穿着方式区分：

```
星见学园校服（统一设计）：
- 男生：深蓝色西装外套 + 白衬衫 + 深色领带 + 灰色长裤
- 女生：深蓝色西装外套 + 白衬衫 + 深色蝴蝶结 + 灰色百褶裙
- 季节差分：
  - 夏季：白色短袖衬衫（去掉外套），女生裙长不变
  - 冬季：加围巾/大衣（深色）
- 共享元素：校徽（左胸口，银色刺绣）
```

### 表情差分统一标准

所有角色的表情差分遵循统一命名和规格：

| 差分名 | 统一定义 | 通用画法要点 |

| smile | 开心微笑 | 嘴角上扬，眼微弯 |

| surprise | 惊讶 | 眼睁大，嘴微张 |
| shy | 害羞 | 脸红，低头或侧脸 |
| think | 思考 | 眼看向侧上方，手托腮或抵唇 |

### 角色差异化设计规范

在统一画风下，通过以下维度区分角色：

```
差异化维度（按重要性排序）：
1. 发型/发色：最直观的区分要素
2. 瞳色/瞳孔形状：第二区分要素
3. 体型/身高：站姿时的视觉差异
4. 配饰/穿着方式：校服下的个人特色
5. 表情习惯：角色特有的微表情
6. 氛围/气场：整体给人的感觉（通过色调/光影体现）
```

## 角色资源复用性策略

> ⚠️ 核心目标：减少重复制作，最大化跨角色线资源复用。

### 复用层级

```
Level 1：全局共享资源（所有角色线共用）
  - 校园背景图（教室/美术教室/体育馆/天台/图书馆）
  - 城市背景图（河岸/商店街/展览馆/展望公园）
  - 天气差分（晴/雨/雪/夜）
  - UI元素（对话框/选项面板/手机界面）
  - 校服基础模板（男女各一套，仅换色/配饰）

Level 2：跨角色线复用资源（多条线共用）
  - 校服立绘模板（同性别角色共享基础骨架）
  - 通用表情差分（normal/smile/sad等标准表情）
  - 共通事件背景（文化祭/修学旅行/卒业式）

Level 3：角色线专属资源（仅当前线使用）
  - 角色便服/特殊服装立绘
  - 角色特有表情差分
  - CG（角色专属场景图）
  - 角色专属背景（如花曦的画展/文陆的天文台）
```

### 校服立绘模板复用策略

```
策略：同性别角色共享校服立绘骨架，仅修改差异化部分

男生校服模板：
  共享：西装外套轮廓 + 衬衫 + 领带 + 长裤 + 姿势
  差异化：发型/发色 + 瞳色 + 体型微调 + 配饰 + 表情

女生校服模板：
  共享：西装外套轮廓 + 衬衫 + 蝴蝶结 + 百褶裙 + 姿势
  差异化：发型/发色 + 瞳色 + 体型微调 + 配饰 + 领口状态 + 表情

制作流程：
  1. 制作基础模板（男女各一套，分层PSD）
  2. 分层结构：身体层 + 头发层 + 表情层 + 配饰层 + 服装细节层
  3. 新角色：复制模板 → 替换差异化层 → 微调整体
  4. 预计节省：每新增同性别角色节省60-70%立绘制作量
```

### 表情差分复用策略

```
策略：通用表情复用骨架，角色特有表情单独制作

通用表情（6种，所有角色共享画法规范）：

角色特有表情（每角色1-3种，单独设计）：

  于桦：distant_gaze（远目）/ gentle_sad（温柔悲伤）
```

### 背景图复用策略

```
策略：校园/城市背景全局共享，仅角色专属场景单独制作

  校园5张：教室/美术教室/体育馆/天台/图书馆

  花曦：画展内部 / 舞台后台
```

### CG复用策略

```
策略：CG不可跨线复用（角色专属），但差分CG可复用构图

CG差分复用：
  同一CG场景 → 不同表情/姿势差分 = 共享构图+背景，仅重画角色部分
  例：CG_HX_06（天台·雨）→ 差分A（蹲·哭）/ 差分B（站·望天）
  节省：差分CG仅重画角色层，背景层复用

CG构图模板复用：
  高频构图类型可跨线参考（非直接复用）：
  - 窗边侧脸型：初见CG通用构图
  - 天台背影型：高潮CG通用构图
  - 对视特写型：告白CG通用构图
```
