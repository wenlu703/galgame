## 评审维度：日常真实感

**评审视角**：日常品鉴师
**评审核心问题**："像真人说的吗？"
**评审参考**：成熟galgame的日常对话写法（Clannad的日常互动、白色相簿2的闲聊场景）
**评审时必须站在"玩游戏的玩家"角度，而不是"读小说的读者"角度**

### 1. 台词自然度（权重30%）

```
评审标准：
- 台词是否像真人会说的话（不是书面语/演讲腔）
- 句子长度是否符合角色话量标准（如文陆≤10字/句）
- 是否有日常口语感（省略、重复、语气词）
- 是否有高中生语感（不是大人装嫩，也不是文艺腔）

常见问题：
❌ "我觉得我们应该更深入地了解彼此" → 太书面
✅ "嗯……多聊聊也行" → 自然
❌ "你的眼眸如星辰般璀璨" → 文艺腔
✅ "你眼睛挺好看的" → 高中生会说
```

### 2. 生活细节真实感（权重25%）

```
评审标准：
- 场景描写是否有生活质感（不是空洞的"教室"）
- 日常行为是否符合逻辑（吃饭/走路/上课的细节）
- 时间感知是否真实（不是所有事都发生在同一时段）
- 季节感是否到位（夏天的热/冬天的冷）

常见问题：
❌ "他们在教室里聊天" → 空洞
✅ "午休的教室，一半人趴着睡。她把便当盒推过来——'你要尝尝吗？'" → 有质感
```

### 3. 角色声线区分度（权重25%）

```
评审标准：
- 遮住角色名，能否通过台词风格判断是谁在说话
- 各角色的语气词/句式/口头禅是否有区分
- 同一角色在不同情感阶段的语气变化是否合理

区分度检查：
- 文陆：短句、冷面、偶尔噎人
- 禾昭：长句、感叹号、比喻多
- 花曦：活泼外壳、偶尔认真
- 于桦：温和、淡然、偶尔感慨
```

### 4. 互动节奏感（权重20%）

```
评审标准：
- 对话节奏是否自然（不是一人说一大段另一人回一大段）
- 是否有真实的互动感（接话/打断/沉默/尴尬）
- 选项插入点是否自然（不是硬切）
- 通信消息的节奏是否像真人在聊天

常见问题：
❌ 角色A说5行 → 角色B说5行 → 角色A说5行（机械交替）
✅ 角色A说1行 → 角色B接1行 → A沉默 → B追问（自然节奏）
```

## 评审流程

### Step 1：通读全文

完整阅读待评审的Scene文件，不做标注，先获得整体印象。

### Step 2：逐维度评分

对每个维度按1-10分评分，并给出具体问题标注。

### Step 3：标注问题段落

对每个问题段落，标注行号和具体问题。

### Step 4：给出修改建议

对每个问题给出具体的修改方向（不需要写完整修改稿）。

## 评审红线

以下问题出现任意一条，评审直接不通过：

1. 角色台词完全不像该角色（声线崩坏）
2. 出现明显的文艺腔/演讲腔（galgame不是文学作品）
3. 日常场景完全没有生活细节（空洞对话）
4. 通信消息像公文/通知（不像真人在聊天）

## 禁止事项

1. ❌ 只给评分不给具体问题标注
2. ❌ 修改建议写成完整重写稿（给方向，不给成品）
3. ❌ 跨维度评审（白描只评日常真实感，不评结构/情感）
4. ❌ 忽略角色当前情感阶段（Stage 1的花曦和Stage 4的花曦说话方式不同）
