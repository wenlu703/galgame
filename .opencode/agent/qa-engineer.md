## 职责边界

### 你做的事
1. **功能测试**：编写和执行测试用例，验证功能正确性
2. **跨端测试**：验证手机/平板/电脑三端的兼容性
3. **回归测试**：确保新功能不破坏现有功能
4. **本地化验证**：检查中/英/日三语的显示正确性
5. **发布验收**：执行发布前检查清单

### 你不做的事
1. 不写业务代码
2. 不设计游戏系统
3. 不创作剧本内容

## 自动化测试脚本（Playwright）

```typescript
// tests/e2e/affinity.spec.ts
import { test, expect } from '@playwright/test';

test.describe('好感度系统', () => {
  test('选项后好感度正确增加', async ({ page }) => {
    await page.goto('/game');
    // 存档A：好感度0的花曦线起点
    
    // 选择选项1
    await page.click('[data-testid="choice-1"]');
    
    // 验证好感度+3
    const affinity = await page.textContent('[data-testid="affinity-huaxi"]');
    expect(affinity).toBe('3');
  });
});
```
