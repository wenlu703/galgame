## 1. 城市场景设计（12处）

星见市以星见学园为中心，向外辐射12处功能场景。每处场景包含环境描述、功能定位、视觉风格、关联角色、解锁条件、时间可用性。

#### 场景一：中央商店街（中央商店街）

| 项目 | 内容 |
|------|------|
| **环境描述** | 覆盖着玻璃拱顶的传统商店街，长约300米，两侧是各类店铺——文具屋、书店、甜品店、卡拉OK、游戏中心、拉面店、杂货铺。拱顶下挂着季节性装饰，傍晚时分暖黄色灯光亮起，是放学后学生最常聚集的区域 |
| **功能定位** | 日常社交核心区——放学后闲逛、周末约会、打工、购物 |
| **视觉风格** | 暖色调，玻璃拱顶透光，店铺招牌密集但有序，有生活气息的繁华感 |
| **关联角色** | 禾昭（游戏中心常客）、于桦（书店/卡拉OK）、文陆（文具屋/书店） |
| **解锁条件** | 默认解锁 |
| **时间可用性** | 全天开放；放学后（16:00-19:00）和周末为高峰时段 |

**商店街内子场景：**

| 子场景 | 功能 | 关联角色 |
|--------|------|----------|
| 游戏中心 | 电竞对战、夹娃娃机 | 禾昭 |
| 书店·星见堂 | 买书、看书、偶遇 | 文陆、于桦 |
| 甜品店·Sugar | 下午茶、聊天 | 全角色 |
| 卡拉OK·Melody | 唱歌聚会 | 于桦、禾昭 |
| 文具屋·手创馆 | 手工材料、画材 | 花曦 |
| 拉面店·一星 | 快餐、打工 | — |

#### 场景二：星见市立展览馆（星见市立美术馆）

| 项目 | 内容 |
|------|------|
| **环境描述** | 两层楼的现代建筑，白色外墙配大面积玻璃幕墙。常设展厅为本地艺术家作品，特展厅每季度更换主题。一楼有咖啡厅和纪念品商店。大厅挑高，自然光从天窗洒下，安静而庄重 |
| **功能定位** | 文化体验区——看展、志愿者活动、艺术启蒙。花曦线的核心校外场景 |
| **视觉风格** | 冷色调为主，白色+灰色+原木色，干净通透，有艺术氛围 |
| **关联角色** | 花曦（美术方向核心场景） |
| **解锁条件** | 花曦好感≥30时解锁（花曦主动提起"你知道市里有个展览馆吗？"） |
| **时间可用性** | 10:00-18:00（周一休馆）；周末特展人较多 |

#### 场景三：星见 Cinema（星见影院）

| 项目 | 内容 |
|------|------|
| **环境描述** | 车站前通旁的两层小型影院，共3个放映厅。一楼售票处和卖品部，二楼放映厅。休息区有几张复古沙发和电影海报墙。整体氛围怀旧而温馨，不是大型连锁影院，是那种"小镇上只有一家"的感觉 |
| **功能定位** | 约会/社交场景——看电影、打工。galgame经典约会场景 |
| **视觉风格** | 暖色调，深红+金色，复古电影海报，昏暗而浪漫 |
| **关联角色** | 于桦（电影爱好者） |
| **解锁条件** | 默认解锁 |
| **时间可用性** | 10:00-22:00；周末晚间为约会高峰 |

#### 场景四：星见湾游乐园（星见湾游乐园）

| 项目 | 内容 |
|------|------|
| **环境描述** | 海滨小型游乐园，摩天轮是地标。有过山车、旋转木马、碰碰车、射击场等经典项目。海风吹过，远处能看到海平线。傍晚时分摩天轮亮灯，整个园区被暖色灯光笼罩。入口处有棉花糖和刨冰摊位 |
| **功能定位** | 高级约会/集体活动场景——学园祭后的二次会、修学旅行自由时间、告白场景 |
| **视觉风格** | 明亮活泼，蓝天+彩色设施，傍晚转暖色灯光，有梦幻感 |
| **关联角色** | 禾昭（过山车爱好者） |
| **解锁条件** | 默认解锁 |
| **时间可用性** | 10:00-20:00（冬季缩短至17:00）；周末及假期为主 |

#### 场景五：星见台展望公园（星见台展望公园）

| 项目 | 内容 |
|------|------|
| **环境描述** | 位于城市西侧小山丘顶的公园，需爬一段缓坡或乘巴士到达。公园不大，有一片草坪、几棵大树、长椅和一个小型观景台。从这里可以俯瞰整个星见市——白天能看到星见学园的校舍和星见川，夜晚则是一片灯海，星见川的倒影如碎金。风比市区大，夏天有蝉鸣，秋天有红叶 |
| **功能定位** | 安静独处/深度对话场景——告白、坦白心事、看夜景 |
| **视觉风格** | 白天清新（绿+蓝），黄昏温暖（橙+紫），夜晚浪漫（深蓝+灯光金点） |
| **关联角色** | 花曦（坦白/告白场景）、文陆（深度对话场景） |
| **解锁条件** | 默认解锁 |
| **时间可用性** | 全天开放；傍晚至夜间为最佳时段 |

#### 场景六：星见市立图书馆（星见市立图书馆）

| 项目 | 内容 |
|------|------|
| **环境描述** | 三层楼的公共图书馆，外观是红砖建筑，带有一点大正时代的复古感。一楼是儿童区和期刊室，二楼是开架阅览室和自习区，三楼是特别收藏室和安静自习隔间。自习区靠窗的位置最受欢迎，下午阳光斜着照进来，能看到窗外的银杏树。馆内极其安静，只有翻书声和空调的嗡鸣 |
| **功能定位** | 学习/独处/偶遇场景——自习、查资料、与学霸型角色偶遇。文陆线的核心校外场景 |
| **视觉风格** | 沉稳暖色调，红砖+深木色+暖黄灯光，有知性氛围 |
| **关联角色** | 文陆（周末自习据点） |
| **解锁条件** | 文陆好感≥30时解锁（文陆无意中提到"市图书馆的自习室很安静"） |
| **时间可用性** | 9:00-21:00（周三休馆）；考试季座位紧张 |

#### 场景七：星见市立动物园（星见市立动物园）

| 项目 | 内容 |
|------|------|

#### 场景八：星见川河岸公园（星见川河川公园）

## 2. 场景解锁系统

城市场景分为**默认解锁**和**角色解锁**两类。角色解锁的场景与该角色的核心人设强相关，解锁时角色会主动提及该地点，形成自然的叙事引导。

| 场景 | 解锁类型 | 解锁条件 | 解锁触发方式 |
|------|----------|----------|-------------|
| 中央商店街 | 默认解锁 | — | 游戏开始即可前往 |
| 星见 Cinema | 默认解锁 | — | 游戏开始即可前往 |
| 星见湾游乐园 | 默认解锁 | — | 游戏开始即可前往 |
| 星见台展望公园 | 默认解锁 | — | 游戏开始即可前往 |
| 星见川河岸公园 | 默认解锁 | — | 游戏开始即可前往 |
| 车站前通 | 默认解锁 | — | 游戏开始即可前往 |
| 住宅区·西 | 默认解锁 | — | 游戏开始即可前往 |
| **星见市立展览馆** | **花曦解锁** | 花曦好感≥30 | 花曦主动提起"你知道市里有个展览馆吗？" |
| **市立图书馆** | **文陆解锁** | 文陆好感≥30 | 文陆无意中提到"市图书馆的自习室很安静" |
| **市立动物园** | **皐月解锁** | 皐月好感≥30 | 皐月兴奋地说"动物园新来了小熊猫！" |
| **住宅区·东** | **花曦/于桦解锁** | 花曦好感≥50或于桦好感≥50 | 送角色回家时解锁 |
| **星见山森林公园** | **剧情解锁** | 修学旅行剧情 | 修学旅行集体活动时解锁 |

**解锁系统的叙事意义**：

1. **防止校园剧情稀释** — 角色解锁机制确保玩家在深入城市之前，已经与角色建立了足够的关系基础。城市不是"随便逛的地方"，而是"和某个人一起去的地方"
2. **强化角色-场景绑定** — 每个角色都有自己的"城市领地"，去展览馆就想到花曦，去图书馆就想到文陆，去动物园就想到皐月。场景成为角色的延伸
3. **提供探索动力** — 玩家为了解锁新场景，会主动提升角色好感度，形成"提升好感→解锁场景→获得新互动→提升好感"的正循环

## 3. 天气系统

游戏内设6种天气类型，影响场景可用性和角色通信内容。

#### 天气类型与游戏内效果

| 天气 | 出现概率 | 对场景的影响 | 对通信的影响 |
|------|----------|-------------|-------------|
| ☀️ 晴 | 40% | 所有场景正常开放；河岸公园和展望公园为最佳状态 | — |
| ⛅ 多云 | 25% | 所有场景正常开放 | — |
| 🌧️ 雨 | 20% | 户外场景氛围变化（河岸打伞散步、商店街拱顶避雨）；户内场景人流量增加 | 角色可能发来天气相关消息（花曦："你带伞了吗？"） |
| 🌧️ 大雨 | 5% | 户外场景不可用；游乐园闭园 | 角色发来关心消息；于桦可能约去Cinema |
| ❄️ 雪 | 5% | 户外场景氛围变化（河岸雪景、展望公园银装）；游乐园缩短营业 | 角色发来雪景照片或约去玩雪 |
| 🌸 花粉 | 5% | 春季限定；户内场景人流量增加 | 角色吐槽花粉（文陆："今天又忘了带口罩。"） |

#### 天气与剧情的联动

- **雨天**是galgame经典场景——共伞、避雨、在屋檐下等待雨停。河岸公园和商店街拱顶下是最佳雨天场景
- **雪天**是告白/深度对话的催化天气——展望公园雪景是星见市最浪漫的场景之一
- **大雨天**强制将活动收束到户内场景（Cinema、图书馆、展览馆），自然形成约会场景
