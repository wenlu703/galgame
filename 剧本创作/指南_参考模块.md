## 1. 写作团队

为避免单一写手导致剧情风格单调，剧本由 **3位不同专长的写手** 各自创作关键部分，再由 **主编** 统筹合成为统一风格的终稿。

#### 写手分工

| 角色 | 专长领域 | 最适合处理的场景 |
|------|---------|----------------|
| **主编**（统筹） | 整体架构、风格统一、终稿合成 | 大纲设计、分镜规划、评审意见整合、最终定稿 |
| **「素心」** | 情感细腻度、内心戏 | 感情升温的关键场景、角色脆弱时刻、浪漫/感人段落 |
| **「墨骨」** | 戏剧张力、热血场面 | 体育比赛、游戏对战、冲突转折、高潮爆发场景 |
| **「白描」** | 日常质感和台词 | 日常碎片事件、课间对话、生活细节打磨、角色声音区分 |

#### 写作流程

```
主编产出剧情大纲（事件序列+核心冲突+关键选择节点）
  → 素心写手负责感情戏场景
  → 墨骨写手负责张力/动作/高潮场景    （并行创作）
  → 白描写手负责日常对话和过渡场景
    → 主编统合全部素材，润色风格一致性，产出完整剧本终稿
      → 进入7人评审团 × 3轮循环
```

## 2. 评审机制

所有剧本必须经过 **7 位评审的集体审批 × 完整3轮修改循环**后才能呈现给用户。一轮的定义是：评审出意见 → 写手修改。必须完成三轮。

#### 7 位评审角色

| # | 评审角色 | 评审维度 | 核心问题 |
|---|---------|---------|---------|
| ① | **人设警探** | 角色一致性 | 角色言行是否符合设定？有OOC吗？ |
| ② | **逻辑法官** | 剧情逻辑 | 因果自洽吗？时间线有漏洞吗？ |
| ③ | **真实感质检员** | 生活真实度 | 场景/对话贴合中国高中实际吗？ |
| ④ | **情感捕手** | 情感感染力 | 有共鸣吗？感情发展自然吗？ |
| ⑤ | **选择架构师** | 分支质量 | 选项有意义吗？有假选项吗？ |
| ⑥ | **台词质检员** | 台词质量 | 角色说话方式有区分度吗？自然吗？ |
| ⑦ | **成长线监理** | 递进节奏 | 8~10事件的渐进感够吗？有跳跃吗？ |

#### 完整3轮循环工作流

```
第一轮（初稿 → 修改）：
  主编+3写手产出完整剧本v1
    → 7位评审并行审核，输出评审意见（附评分：✅/⚠️/❌）
      → 主编+3写手根据全部意见修改，产出v2

第二轮（复审 → 再改）：
   → 7位评审复审v2（重点关注上轮❌问题是否解决）
     → 主编+3写手再次修改，产出v3

第三轮（终审 → 定稿）：
   → 7位评审终审v3（确认所有问题已解决）
     → 主编+3写手最终修订，产出v_final
       → 提交给用户审阅
         → 用户确认 → 写入「二、剧情记录」
         → 用户要求修改 → 从第一轮重新开始
```

#### 评审守则

- 每位评审只关注自己的维度，不越界
- 评审意见必须具体到**哪一幕/哪句台词/哪个选择**
- 评分使用三档制：✅ 达标 / ⚠️ 小问题 / ❌ 需要重写
- 任何一项若为 ❌，本轮修改时该问题必须解掉才能进下一轮
- 三轮循环不可省略、不可缩减、不可合并

## 3. 手机通信系统（完整版）

所有可攻略NPC必须拥有完整的手机通信系统，通信内容是剧情叙事的重要组成部分。

#### 情感阶段划分与信息量要求

每位可攻略NPC的通信信息库总量为**20条**，分散到各好感度阶段，数量根据角色性格灵活分配。核心原则：**信息内容必须符合角色人设和当前情感阶段。**

| 情感阶段 | 好感度区间 | 建议信息量 | 信息特征 |
|----------|-----------|-----------|----------|
| **初始** | 0-19 | 3-4条 | 礼貌性、事务性、试探性。角色保持社交距离，消息简短 |
| **熟悉** | 20-49 | 4-5条 | 日常分享、轻松闲聊、偶尔吐槽。角色开始主动发消息 |
| **好感** | 50-99 | 4-5条 | 主动关心、分享私事、暗示性话题。消息频率增加 |
| **喜欢** | 100-199 | 4-5条 | 依赖性消息、深夜聊天、情绪化表达。消息成为情感出口 |
| **爱** | 200+ | 3-4条 | 坦白、承诺、未来规划。消息是最私密的对话空间 |

#### 消息字段规范

每条信息必须包含以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| ID | 唯一标识 | `msg_huaxi_07` |
| 角色ID | 关联角色 | `huaxi` |
| 阶段 | 情感阶段 | `familiar` |
| 触发条件 | 何时可触发 | `好感度≥20 AND 周末 AND 天气=晴` |
| 发送者 | 角色或主角 | `character` |
| 内容 | 消息正文 | "今天美术部的学姐说我的配色有进步！" |
| 回复选项 | 玩家可选回复（0-3个） | ["太好了！", "你画了什么？", （不回复）] |
| 回复分支 | 不同回复的后续 | 选项1→好感+3；选项2→解锁新话题；选项3→无 |
| 好感度变化 | 触发后好感度变化 | +2 |
| 关联剧情 | 是否关联主线事件 | `Act3_天台救援` |

#### 聊天对话节奏规范

与角色的网络聊天对话需遵循以下节奏控制规则：

1. **角色分享长度** — 角色单次分享（从角色发起话题到该话题自然结束）应在**8~15句内**完成，不宜过长导致玩家失去参与感，也不宜过短导致内容空洞
2. **主角选项次数** — 每次聊天对话中，主角的选项交互应在**3次以内**结束，避免选项过多导致节奏拖沓。选项设计应遵循"无假选项"原则，每个选项都有独特的叙事反馈
3. **颜文字使用规范** — 角色可附带符合其人设的颜文字，但必须遵循角色性格：
   - 活泼型角色（禾昭、皐月、小樱等）：可使用丰富颜文字（✧、(≧▽≦)、♪等）
   - 温柔型角色（花曦等）：偶尔使用简单颜文字（♪、☆、…等）
   - 冷酷/内敛型角色（文陆等）：**不允许使用颜文字**，保持文字的克制感

**聊天节奏示例（花曦·好感阶段）：**

```
花曦：今天美术部画了静物，苹果和橙子
花曦：学姐说我的配色有进步♪
  → [选项1] 你画了什么？ / [选项2] 进步了多少？ / [选项3] （发个赞）
花曦：就是普通的静物啦，不过这次光影处理比上次好
花曦：进步嘛……大概从"不太行"进步到"还行"？哈哈
  → [选项1] 那就是很大的进步 / [选项2] 下次让我看看
花曦：嗯！下次展览馆有新展的话一起去吧
花曦：先不说了，我妈叫我吃饭……明天见☆
```

#### 通信触发机制

**被动接收：**

| 触发类型 | 说明 | 示例 |
|----------|------|------|
| **时间节点触发** | 特定日期/时间段自动触发 | 生日当天收到祝福消息 |
| **剧情进度触发** | 主线剧情推进后触发 | 天台事件后花曦的消息 |
| **好感度阈值触发** | 好感度达到新阶段时触发 | 好感度突破50时角色发来"最近总觉得跟你聊天很开心" |
| **场景触发** | 角色在特定城市场景时触发 | 于桦在Cinema看完电影后发来感想 |
| **天气触发** | 特定天气条件下触发 | 下雨天花曦发"你带伞了吗？" |
| **随机触发** | 满足基础条件后随机触发 | 日常闲聊消息池 |

**主动发送：**

| 条件 | 说明 | 限制 |
|------|------|------|
| 好感度≥20 | 解锁主动发消息功能 | 每个角色每天最多主动发起1次 |
| 好感度≥100 | 解锁"约出来"功能 | 需对应城市场景已解锁 |
| 特定话题解锁 | 通过剧情/活动解锁新话题 | 已解锁话题可反复使用但好感度只加一次 |

**预约功能：**

| 功能 | 说明 | 示例 |
|------|------|------|
| **活动预约** | NPC通过消息邀请主角参加特定活动 | 花曦："周六展览馆有新展，要不要一起去？" |
| **时间预约** | NPC预约特定时间进行剧情事件 | 于桦："下周五放学后有空吗？想跟你说件事。" |
| **地点预约** | NPC指定城市场景见面 | 文陆："明天中午，图书馆。有东西给你。" |

**预约机制规则：**
- 预约消息在约定时间前1个游戏日内到达
- 玩家可选择"答应"/"改时间"/"拒绝"
- 答应后自动在日程中标记，到时间时系统提醒
- 拒绝会降低好感度（-3至-10，视角色性格而定）
- 改时间需角色同意（好感度越高越容易同意）

#### 手机通信系统创作规范（最高优先级）

> 以下条款作为所有剧情创作必须遵守的基础规则：

1. 所有可攻略NPC必须拥有完整的通信信息库（总量20条，分散到各情感阶段）
2. 关键剧情事件发生后，必须设计至少1条相关的手机通信内容
3. 角色在城市场景活动时，必须设计至少1条场景关联通信内容
4. 通信内容的语气、长度、发送时间必须符合角色人设和当前情感阶段
5. 通信系统中的回复选项必须遵循"无假选项"原则——每个选项都应有独特的叙事后果
6. 预约功能仅用于关键剧情事件和高级约会场景，不得滥用
7. 通信信息不得包含与当前剧情进度矛盾的内容（如角色在住院期间发来日常闲聊）
8. 角色单次分享应在8~15句内结束，主角选项交互在3次以内结束；颜文字使用必须符合角色人设，冷酷/内敛型角色不允许使用颜文字
9. NPC个人线剧情创作时，每个阶段必须同时包含该阶段的网络消息对话信息和关键剧情选择信息，三者齐全方可视为完整的创作产出

注：9条强制条款同时存在于基础模块中，此处保留完整版作为参考

### 5.2 剧情线草案——文陆篇《他的另一面》

> **核心设计思路**：文陆的剧情线以"反差的递进"为主线——先展现冷的表面，再揭示热的内心，最后触及柔软的深处。主角既是"被帮助者"，也是最终唯一被允许走进他世界的人。

#### Act 1：初次接触——"学习委员的职责"

**时间**：高一下学期，开学第二周，雾海第一次走进六班教室

班主任带雾海进班时，全班的目光落在这个陌生的男生身上。文陆作为学习委员，被安排负责帮助新同学适应——"毕竟你落了整个学期的课，文陆会帮你整理笔记和进度。"

最初的几次接触完全是事务性的。文陆递过来一叠整理好的笔记本，语气平淡："这学期的笔记都在这里，你先看，不懂的划下来，中午我讲。"

主角感觉到这不是冷漠，而是一种——效率。文陆用最简洁的方式完成"帮助新同学"这个任务，不寒暄、不拖泥带水。

但正是在这种简洁中，主角注意到细节：笔记用不同颜色的笔标注了重点和难点，页脚还写了每章的自我检测题答案页码。这不是"完成任务"的敷衍，而是认真到近乎强迫症的习惯。

▸ "笔记做得很详细，谢谢。"（礼貌回应）
▸ "你每一科都记成这样？不累吗？"（试探性好奇）
▸ "……好。"（默默接受——沉默路线）

#### Act 2：破冰——"一道题的解法"

经过一周的自习，主角对文陆的笔记已经有了基本的了解。某天午休，主角在做一道物理题时卡住了——那是上学期内容的综合题，笔记上有步骤但跳过了中间推导。

文陆刚好回教室拿东西，看到主角在草稿纸上反复划同一道题。他停下，看了几秒。

"第三行这个公式代错了。上学期学的，你没见过所以不知道也正常。"

他在草稿纸上重新写了一遍推导过程。写完后没有马上走，而是等了几秒——确认主角看懂后才离开。

从这天起，文陆偶尔会在午休时主动问一句"有不懂的吗？"——语气依然是平淡的，但这种例行公事般的询问本身，已经在表明他开始在意这个认真的后进生。

#### Act 3：信任的裂缝——"第一次看到他发火"

某天下午，文陆接了一个电话后脸色明显不对。他跟老师请了假提前走。第二天回来时，眼下的青黑遮不住——但他什么都没说，照常上课，照常做笔记，照常高效运转，好像什么都没发生。

禾昭低声跟主角说："别问他。他不想说的时候，问了他也不会说。"

这是主角第一次直观地感受到文陆的"倔强"。他像一台完美的机器，拒绝在任何外人面前露出故障的迹象。但主角注意到，那天文陆的笔记里，有一页的页脚被笔尖戳穿了几个小洞——那是他唯一泄露情绪的痕迹。

#### Act 4：走进——"雨天的伞"

一个雨天。放課後自習结束后突然下起大雨，很多人被困在教学楼。

主角看到文陆站在教学楼门口，没有伞，但神色平静——他打算等雨小一点再冲回去。主角走过去，把伞递给他。

"你家住哪个方向？"

文陆愣了一下——他可能没想到这个平时不怎么说话的新同学会主动过来。报了一个小区名后，发现和主角顺路。

两人打着同一把伞走进雨里。话不多，但这是他们第一次在教室之外的空间相处。

雨声中，文陆忽然开口："你家里……只有你一个人？"

主角简单说了自己的情况——奶奶去世，父母在国外。文陆听完沉默了一段路，然后说："我家里人去年也住院过一段时间。挺难的。"

这是文陆第一次主动提及自己的私事。虽然只是一句话，但在这个雨夜，说出来就已经意味着什么。

#### Act 5：转折——"借我三千块钱"

某天文陆没有来上学。禾昭跟主角说："他家好像出事了。"

第二天文陆回来时神情如常，但主角看到他课间用手机打了很久的电话，回来后在草稿本上写了一串数字又划掉——那是一笔医药费的估算。

午休时，教室里只剩下他们两人。主角犹豫了很久，还是开口了："文陆，如果有需要帮忙的……"

文陆抬头看着他。那种被看穿的瞬间，文陆的第一反应是防御——"没事，我自己能处理。"

但主角没有移开目光。

"嗯。如果需要的话，我在。"

就这么一句。没有追问，没有怜悯，只是表明存在。文陆沉默了很久，然后低下头，低声说了一个数字。

这是文陆第一次——在任何人面前——承认自己撑不住了。

#### Act 6：升华——"伯牙的子期"

此后他们的关系进入了一个新的层次。文陆还是会帮主角补习功课，但不再是单向的"好学生帮助差生"——辅导变成了交流，讨论变成了争论，偶尔还会因为一道题的解法互呛，然后在禾昭的插科打诨中结束战斗。

周末，文陆会偶尔来主角家一起复习。他带来的不仅仅是习题集——有时候是自己做的酱菜（"我妈腌的，让我带点给你"），有时候是一本他最近在看的杂书。

在某一晚复习结束后，文陆收拾书包时，忽然说了一句前言不搭后语的话：

"你转来这个班……挺好的。"

然后他背着书包走了，像什么都没说一样。

---

**初始好感度选项分支示例**：

① Act 2 中，主角对文陆的回应方式会影响关系走向：
- "笔记做得很详细，谢谢。" → 温和发展，文陆以"责任"为起点建立信任
- "你每一科都记成这样？不累吗？" → 幽默敏锐路线，文陆注意到主角的观察力
- "……好。"（沉默）→ 慢热路线，需要更多日常积累才能破冰

② Act 5 中，面对文陆的危机时：
- "如果需要帮忙的话……"（主动开口）→ 直接攻破文陆防线，关系大幅升温
- 默默帮文陆整理好当天各科作业放在他桌上（暗中行动）→ 文陆注意到无声的关心，好感度持续积累

### 5.3 剧情线草案——禾昭篇《球场与兄弟》

> **核心设计思路**：禾昭的线以"动"制"静"——用他阳光的活力来融化主角初来的拘谨，是六班这条线中最早与主角建立轻松关系的角色。剧情以体育活动（篮球赛）和电竞游戏为双主线场景。

#### Act 1：自来熟——"嘿，新同学！"

就在文陆给主角递笔记的那个课间，文陆还没来得及说什么，他同桌就转过头来——一张带着明亮笑容的微黑脸庞。

"诶，你是新来的？我叫禾昭！以后一个班的，有事找我就行！"

禾昭的搭话来得自然而不突兀，好像他们已经认识了很久。他根本不在意主角的沉默寡言，自顾自地说了下去——"你之前是哪个学校的？""你打不打篮球？""中午一起去食堂吧？""我跟你说，三楼的麻辣香锅最好吃——"

文陆在旁边头也不抬地补了一句："他话很多，习惯就好。"

禾昭："喂！"

这个课间，主角什么都没做——只是听禾昭说了十分钟的话。但他发现自己没有觉得厌烦。

#### Act 2：球场——"传球！"

周五体育课，自由活动时间。禾昭抱着篮球跑过来："走，缺个人，凑一队！"

主角还没来得及说"我打得不好"，已经被禾昭拽着往球场走了。

球场上，禾昭完全是另一个人。平时嘻嘻哈哈的他，拿到球的那一刻眼神就变了。跑位、卡人、突破、分球——每一个动作都带着一种不容置疑的专注。但他不是独狼——打进之后第一件事不是自己庆祝，而是转头找传球的队友，伸手拉一把。

"好传！"

打完球，禾昭买了两瓶水，扔给主角一瓶，一屁股坐在旁边草坪上，大口喘气——然后又笑了。

"你打得还不错嘛！下次还来！"

主角接过水，发现禾昭买的是自己平时喝的牌子——他什么时候注意到的？

#### Act 3：游戏之夜——"别送啊兄弟！"

周末晚上。禾昭在Xchat上拉主角打游戏——不是竞技排位，就是大乱斗。开了语音之后，禾昭的声音和平时很不一样——更放松，更随意，偶尔还会因为操作变形发出夸张的惨叫。

"他是不是在对面草丛蹲我——他肯定在蹲我——我上了——啊我死了！"

文陆也安静地挂在语音里（他不打但会听），偶尔在一段沉默后精准地补一刀："你刚才那个操作，我奶奶来都赢了。"

语音里笑成一片。

这是主角第一次感受到——在这个全新的环境里，自己有"可以这样待着"的地方。

#### Act 4：转折——"我想走篮球这条路"

期中考试后，禾昭的成绩出来——中上，稳定，但没有惊喜。班主任找他谈话，委婉地提醒"光靠体育生走不远，学习不能放松"。

那天晚上，禾昭约主角在操场上坐了一会儿。难得安静。

"我跟我爸妈说我想走篮球这条路。"他望着空荡荡的球场，说，"他们说'随便你'——但我听得出他们就是觉得我做不到。"

他低下头，两只手交叉握着。

"但我真的想试试。"禾昭的声音不大，但很坚定。

主角坐在他旁边，没有说话。过了一会儿，他开口："那就试试呗。又不丢人。"

禾昭转过头，看着主角，笑了。跟平时那种阳光的笑不一样——更像是松了一口气。

"谢了，兄弟。"

#### Act 5：升华——"一起赢的"

校际篮球赛。六班在决赛中对阵九班（体育生最多的班）。所有人都觉得六班会输。

比赛还剩最后30秒。六班落后3分。球在禾昭手上。

他突破了两个人，在三分线外起跳——九班的防守球员扑了上来，在空中发生了身体接触。禾昭失去平衡，但在落地之前，他把球传向了底角——那里，主角正站在那里。

不是主角主动跑到的位置——是禾昭在突破之前，用眼神示意他"去底角等着"。

球到手。还有5秒。主角投了出去。

——进了。哨响。绝平。

加时赛后，六班赢了。禾昭冲过来，一把抱住主角，差点把他撞倒："你他妈太牛了——！！"

那天他们一起拍了一张合照。禾昭把照片设成了Xchat聊天背景。

"这是我们赢的。"

※ 这个场景将是禾昭满好感度的关键触发事件之一。

---

**初始好感度选项分支示例**：

① Act 1 中，面对禾昭的热情问候：
- "谢谢，我叫雾海。"（正常回应）→ 自然开启友谊线
- "……你好。"（简短回应）→ 禾昭不会在意，反而觉得"有点酷"
- （点头微笑）→ 沉默回应，但禾昭莫名觉得"这人还挺有意思"

② Act 4 操场夜谈的选择：
- "试试呗，又不丢人。"（鼓励）→ 禾昭感到被支持，关系大幅升温
- "你考虑过如果走不通的话吗？"（理性提醒）→ 禾昭觉得主角是真的在为他考虑，信任加深
- "嗯。"（安静陪伴）→ 无声的陪伴比语言更有力

### 5.4 剧情线草案——于桦篇《屏幕那头的世界》

> **核心设计思路**：于桦是"不在场的在场者"。他的故事大部分发生在主角的手机屏幕上——Xchat群聊天、深夜私聊、偶尔分享的照片。这种距离感本身就是他故事的材质：疏离、观察、渴望连接却又身处别处。直到某个周末或两校活动，他才从屏幕背后走到主角面前。

#### Act 1：群聊里的陌生人

主角刚进班级群那天晚上，一个头像是灰色剪影的头像发了一条消息：

"哟，听说六班来了个新同学？欢迎欢迎！"

"@文陆 @禾昭 你们有没有好好照顾人家啊？"

禾昭秒回："滚，我们好着呢。"

文陆隔了很久回了一个句号。

主角问禾昭这人是谁。禾昭说："于桦，我们初中同学，好兄弟。在私立那边读，不在一个学校。"

那天晚上，于桦私聊了主角——"嗨，我是于桦。别理禾昭那张嘴，他人挺好的。你刚来有什么不习惯的可以问我和文陆。"

于是，主角和于桦的聊天就这样开始了。

#### Act 2：私立中学奇谈

于桦的聊天消息总是很长。他喜欢分享私立中学的"奇人异事"——

"今天我们班那个变态考了满分，对，满分，物理。然后他在座位上哭了，说'才考满分'。才。"

"食堂的饭很难吃，但所有人都十分钟之内吃完回教室刷题了。我怀疑他们把时间管理刻进DNA了。"

"我们班有人在课桌底下贴了一张'生而为人，我很抱歉'的纸条。我假装没看到。"

他的叙述总是幽默的、带着调侃的语气。但在那些搞笑的描述底下，主角逐渐读出了一种说不出的——累。

不是身体的累，是被架在某个期望之上、每天绷着弦的累。

#### Act 3：深夜的坦白

某个周六的深夜。主角睡前看到于桦的Xchat头像亮了一下。点开，是一段没有标点的文字：

"雾海你睡了吗没睡的话能聊两句吗"

那是于桦第一次在深夜发来这样的消息。白天的幽默外壳不见了，他说——

今天他被班主任叫出去谈话了。"以你的底子，应该能考进年级前五十的，但你最近的周测，你自己看——"

他说，他其实很讨厌那个学校。讨厌每天早上六点的跑操，讨厌做不完的试卷，讨厌所有人都在比谁更卷。但他不敢跟家里说转学，因为"这是我爸托了好多关系才进来的"。

他说："有时候我会想，如果我现在还在和文陆禾昭一个学校就好了。至少午饭的时候能笑出来。"

信息停留在那一行。屏幕上方的"对方正在输入……"出现了又消失，出现了又消失。

最终他发来了一条："算了，太晚了，你睡吧。晚安。"

#### Act 4：周末的见面

某个周日，文陆约主角和禾昭出来——"于桦说他这周能出来。"

四个人在市中心见了面。这是主角第一次见到屏幕那头的于桦——一个清瘦的男生，戴黑框眼镜，笑起来的时候眼睛眯成一条缝，看起来很无害。

他们去打了篮球（于桦的球技被禾昭无情嘲笑了一番），然后去吃烧烤。席间于桦吐槽私立中学的笑话，把一桌人都逗笑了。文陆话不多，但一直在给于桦夹菜——"吃你的，别光说话。"

散场的时候，于桦走在主角旁边，跟了一句：

"你今天出来，文陆和禾昭挺高兴的。——我也挺高兴的。"

#### Act 5：转折——"我不想读了"

高一下学期末。分科前的最后一个周末。

于桦在Xchat上发来一条消息：

"雾海，我跟我爸妈说了，我想转学。"

然后是一段60秒的语音。主角点开，听到于桦的声音——没有哭，但带着疲惫：

"我知道我要转去一中，现在得看一中的那边有没有名额——文陆说去问他们班主任了。我爸妈还没松口，但我说了，再不转我真的撑不住了。"

"我从来没觉得自己这么差过。但我也知道，不是我的问题，是这个环境的问题。"

"我跟你说的这些……你别跟别人说。禾昭那张嘴我不放心，就你知道就行了。"

信息停留了很久。然后他又发了一条：

"上次你们出来那天，回去的路上我就在想——其实我想要的东西一直很简单，就是能在吃饭的时候笑得出来。"

---

**关系发展说明**：

于桦的线以"倾诉—理解—连接"为主线。物理距离决定了他和主角的关系推进以交流密度和深度为核心，而非日常互动频率。关键的节点在于：

- 于桦转学结果（是否成功转到一中）——这将直接改变他在故事中的定位和戏份占比
- 两校联合活动的线下重逢——让屏幕关系落地的时刻

**需要你确认的方向**：

于桦最终是否转到一中？这会决定：
- **转学成功** → 于桦成为高二分班后的同校角色，戏份大幅增加，成为日常互动常驻角色
- **留在私立** → 于桦维持"重要的外部链接"定位，故事以线上交流+周末活动为主，两校联合活动成为关键场景

### 5.5 剧情线草案——花曦篇《玻璃墙后的花》（v4）

> **⚠️ 设计说明**：花曦的个人线跨越高一至高三，以文化祭开始以文化祭结束。v4根据最高评审意见进行了全线重写：花曦从高一即以校园传说形式登场（文化祭女团舞）、高二通过文陆禾昭铺垫首次见面、高三跳楼事件重写为三人组救援、领口解扣子取代袖口成为核心视觉象征、高三文化祭花嫁+西装作为核心情感高潮、情感线加强（画画转型期陪伴）、全部选项分支嵌入正文。

（完整剧本共 998 行，详见附录文件 `花曦篇_v4.md`。以下为剧情框架概览：）

**高一阶段**：花曦以校园传说形式登场——文陆禾昭口中"文化祭跳女团舞封神"的学姐，画作获市展银赏。主角未入学时即通过他人叙述建立印象。（对应 Scene 1-1 至 1-3）

**高二阶段**：花曦正式进入主角视野——美术课初见（领口扣子全扣）、高二文化祭古风舞惊艳全场、美术教室的"苹果对话"解锁第一次深度交流。领口解扣子在主角前开始松动。（对应 Scene 2-1 至 2-4）

**高三·坠落与拯救**：模拟考失利→被训→失联→老师组织搜索→文陆+禾昭+主角三人组在天台找到花曦→老师吸引注意力→三人上前拦下→救下。主角第一次正面参与花曦的生命危机。（对应 Scene 3-1 至 3-3）

**高三·文化祭高潮**：花曦穿花嫁，之前送主角一套绣有"花见"的西装。主角穿→爱情线（拥抱/带走画/回赠画三个结局）；没穿→友情线（遗憾/希望/温暖三个结局）。以文化祭开始，以文化祭结束。（对应 Scene 5-1 至 5-4）
