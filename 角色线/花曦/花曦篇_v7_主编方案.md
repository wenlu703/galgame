## 二、v7重构总方针

### 2.1 核心目标：让花曦成为玩家真心喜欢的角色

**"喜欢"≠"同情"**。v6的核心偏差在于：玩家对花曦的情感更多是"心疼"而非"心动"。重构必须解决——

| 维度 | v6现状 | v7目标 |
|------|--------|--------|
| 初印象 | 完美优等生·距离感 | 有魅力的活人·想靠近 |
| 互动感 | 安静内敛·被动接受 | 有温度·有小任性·会主动 |
| 危机感 | 突然坠落·令人心疼 | 逐渐崩塌·令人揪心 |
| 恢复感 | 被主角拯救 | 自己站起来·主角是见证者 |
| 结局感 | 温馨但平淡 | 震撼且余韵悠长 |

**五条让玩家爱上花曦的路径**：
1. **魅力先行**——先让玩家被她的活泼与才华吸引，再展示脆弱
2. **对等关系**——花曦也在影响主角，不是单向拯救
3. **主动成长**——破茧是花曦自己的选择，主角是"在她选择时站在旁边的人"
4. **细节记忆**——糖纸、扣子、云的颜色……每个细节都是"只有我们知道"的私密语言
5. **情感递进**——每一次亲密都比上一次更深，直到终章的"不再需要扣子"

### 2.2 重构原则

1. **不推翻已审批内容**：Scene 1合成方案修改版、Scene 2最终版（2-1~2-5）为已审批内容，作为参考但不硬性约束。v7修改从Scene 3-1起。
2. **架构级蓝图**：本方案提供方向与框架，具体文本由剧本架构师执行。
3. **情感弧线必须递增**：每个Scene的情感峰值必须≥前一个同量级Scene，全线最高点必须在终章。
4. **规则零容忍**：R-1/R-2类规则违反必须修复，不可保留。

### 2.3 保留与变更总览

| 内容 | 决策 | 理由 |
|------|------|------|
| Stage 1（Scene 1-1/1-2） | **保留** | 已审批，结构完整 |
| Stage 2（Scene 2-1~2-5） | **保留** | 已审批，2-4排练/2-5一舞为全线最佳Scene |
| Stage 3（Scene 3-1~3-5） | **重构** | 过短过甜，需增加深度与张力 |
| Stage 4（Scene 4-1~4-7） | **重构** | 时间断层+规则违反+被动恢复，需全面调整 |
| Stage 5（Scene 5-1~5-3） | **重构** | 规则违反+高潮不足+友情线单薄 |
| 附录（选项表/扣子轨迹/消息记录） | **同步更新** | 随Scene重构更新 |

## 三、v7各Stage修改方向

已审批内容，不修改。

### 3.2 Stage 2·接近 —— 保留

已审批内容，不修改。

**备注**：Scene 2-4（排练）和2-5（一舞）是v6最出色的两个Scene，它们成功建立了花曦"有才华+有深度"的形象。v7需在此基础上，在Stage 3中展示她的"有温度"。

**一处可优化点（非强制）**：msg_04（花曦第一条消息）语气偏正式，可考虑在v7中微调为更活泼的语气，体现"活泼外壳"。

### 3.3 Stage 3·开启 —— 重构

**核心问题**：v6的Stage 3是"甜蜜日常合集"，5个Scene缺乏戏剧递进，花曦的"活泼外壳"几乎不可见。

**v7方向：从"甜蜜日常"到"信任建立"**

#### 3.3.1 整体定位

Stage 3不应只是"甜蜜"——它是花曦从"对所有人笑"到"只对你真笑"的转变过程。每个Scene都应推进一层信任：

```
3-1 第一道门 → 3-2 放松的试探 → 3-3 私密的分享 → 3-4 主动的邀约 → 3-5 信任的交付
```

#### 3.3.2 各Scene修改方向

| Scene | v6问题 | v7方向 | 关键改动 |
|-------|--------|--------|---------|
| **3-1 展览馆** | 情感峰值偏低（~5.5），从2-5的高潮跌落太多 | 提升情感密度，展览馆应成为"二人世界的第一次" | 增加花曦在展览馆中的活泼表现（对画作的热情评论、小吐槽、主动分享）；"碎掉的光"对话保留但后置为Scene高潮 |
| **3-2 河岸·放学后** | 花曦提到母亲电话后迅速转移话题，缺乏情感层次 | 保留"转移话题"设计，但增加花曦转移话题时的"小狡黠"——她不是在逃避，是在测试"你会不会追问" | 增加花曦的活泼细节（如河岸上踩落叶、对鸭子的评论）；扣子轨迹阶段4在此体现 |
| **3-3 图书馆·糖纸** | 糖纸初现的冲击力不足 | 强化"被看到秘密"的紧张感与甜蜜感 | 花曦被看到糖纸时的反应应更丰富——从慌张到害羞到"你要吗"的小小勇敢；增加花曦对主角的"反向观察"（她也在看主角） |
| **3-4 周末·手创馆** | 花曦提到家庭压力的方式过于直接 | 改为更含蓄的透露——通过画画的行为而非直接倾诉 | 花曦在手创馆应更活泼（试颜料时的兴奋、对画材的如数家珍）；家庭压力通过"画云时停笔"等行为暗示，不直说 |
| **3-5 糖纸的秘密** | 作为Stage 3高潮，情感峰值（~6.5）不够高 | 提升为Stage 3的真正高潮，情感峰值目标~7.5 | 糖纸的完整故事应更有冲击力；花曦说出"每一颗糖纸都是一个我想记住的瞬间"时，应伴随一个更深的信任交付行为（如：把课本里所有糖纸给主角看）；扣子轨迹阶段5在此体现 |

#### 3.3.3 Stage 3新增要素

1. **花曦的"小任性"**：至少1个Scene中花曦表现出小小的任性或固执（如：坚持要去某家店、对某幅画的小执念），让玩家感受到她不是"完美无缺的好学生"，而是有自己小脾气的活人
2. **花曦对主角的"反向影响"**：至少1个Scene中花曦帮助主角解决一个问题（如：帮主角改善画技、帮主角理解某个课程内容），建立关系的对等性
3. **Xchat中的活泼感**：Stage 3的Xchat消息应明显比Stage 2更活泼、更主动，颜文字开始出现（♪☆），语气从"礼貌"转为"亲昵"

### 3.4 Stage 4·深入 —— 重构（重点）

**核心问题**：14个月时间断层 + 文陆/禾昭违规出场 + 花曦过于被动 + 恢复弧过顺。

**v7方向：从"突然坠落"到"缓慢崩塌与自我重建"**

#### 3.4.1 时间线重构

**v6时间线（有问题）**：
```
4-1 高一·12月 → 4-2 高一·1月 → 4-3 高一·3月 → [14个月空白] → 4-4 高二·5月
```

**v7时间线（修复后）**：
```
4-1 高一·12月 → 4-2 高一·1月 → 4-3 高一·3月 → 4-3b 高一·6月 → [暑假Xchat过渡] → 4-3c 高二·4月 → 4-4 高二·5月
```

新增Scene 4-3b和4-3c的功能：
- **4-3b（高一·6月·体育祭前后）**：花曦在体育祭上的表现——她仍然在跳，但跳完后的笑容比文化祭时浅了。主角注意到她的变化。日常缓冲事件。
- **4-3c（高二·4月·新学期）**：新学期开始，花曦明显变瘦了，画室里她的位置有时是空的。她开始缺席美术课。能量明显下降。

这两个Scene填补了时间断层，让天台事件成为"长期恶化的终点"而非"突然的坠落"。

#### 3.4.2 Scene 4-4重构——移除文陆/禾昭

**v6**：禾昭告知花曦情况 → 文陆分析 → 众人分头寻找 → 天台找到花曦 → 三人围住她

**v7方向**：主角独自发现花曦的异常 → 主角独自寻找 → 主角独自在天台找到花曦

具体调整：
- 删除禾昭/文陆的信息提供角色。改为：主角在走廊遇到花曦→注意到她的异常→下午发现她没来上课→独自寻找
- 删除"班主任使眼色+三人围住"的群戏。改为：主角独自走到花曦身边——这个时刻只属于两个人
- 班主任可以保留，但仅作为"确认花曦安全"的后台角色，不参与核心情感场景
- **选项保留**：蹲在旁边/伸出手/"雨停了就回去"三个选项，但反应更私密化

#### 3.4.3 恢复弧重构——花曦的主动性

**v6问题**：4-6中主角包办了问投稿、联系美术老师，花曦仅是"被帮助者"。

**v7方向**：

| Scene | v6 | v7 |
|-------|----|----|
| 4-5 陪伴 | 主角每天去画室陪花曦，花曦被动接受 | 花曦主动发消息邀请主角来画室——"你来的话，我可能画得出来"；恢复过程中花曦有反复（某天突然画不出来比之前更严重），不是线性好转 |
| 4-6 帮助 | 主角去展览馆问投稿、联系美术老师 | 花曦自己决定参加画展，主角帮她的是"陪她去展览馆交材料"和"帮她搬画"等体力/陪伴型支持；美术老师的推荐由花曦自己的画作质量赢得，不是主角争取的 |
| 4-7 破茧 | 花曦站在画前等主角来 | 增加花曦在画展前的紧张与自我怀疑（"如果没人来看怎么办""如果我爸还是不走进来怎么办"），让破茧更有重量；父母出现的处理保留，但增加花曦的主动回应——她不是被动地"他来了就够了"，而是"我要去跟他说话" |

#### 3.4.4 恢复弧中的挫折设计

v6的恢复弧过于顺利（画不出来→陪伴→帮助→画展成功）。v7应增加：

1. **4-5中的反复**：花曦某天突然比之前更画不出来，主角感到无力——这是真实的恢复过程
2. **4-6中的选择压力**：花曦决定参加画展后，面临"画什么"的焦虑，不是主角一句话就解决了
3. **4-7前的不确定**：画展开幕前夜，花曦给主角发消息说"我好害怕明天没有人来"——这不是脆弱，是真实的紧张

### 3.5 Stage 5·终局 —— 重构

**核心问题**：5-2.3违规出场 + 抉择重量不足 + 爱情线后段平淡 + 友情线单薄。

**v7方向：从"温馨尾声"到"情感最高潮"**

#### 3.5.1 Scene 5-1 抉择——增加重量

v6的5-1抉择场景偏轻——花曦问"以后你还会在吗"，主角回答，结束。

v7应增加：
- 抉择前的铺垫：花曦在画室里不是"在等主角"，而是在画一幅画——画的是过去两年和主角在一起的场景。她一边画一边说："我画了两年——终于画到这一页了。"
- 抉择的语境深化：不是简单的"你在不在"，而是"我接下来要走的路很难，你愿不愿意跟我一起走"
- 抉择后的情感爆发：无论选哪条线，花曦都应有更强烈的情感反应（不是微笑+握手，而是真正的"被击中"）

#### 3.5.2 Scene 5-2 告白——提升为全线情感次高点

v6的告白场景（~7.0）情感峰值不够高。v7目标：~8.5。

关键提升点：
- 告白不应只是花曦单方面说"以后让我画你"——应有一个"双向确认"的时刻
- 扣子全解的动作应有更深的铺垫——不是突然解开，而是花曦在告白前有一个"手放在扣子上"的停顿，然后主角说了一句话（或做了一个动作），她才解开
- 告白后的沉默应更有重量——不是"安静得只剩呼吸"，而是两个人都知道"回不去了"的那种安静

#### 3.5.3 Scene 5-2.3 重写——移除文陆/禾昭

**v6**：四人同行·河岸·关系公开

**v7方向**：改为二人场景——花曦和主角回到河岸（他们第一次单独聊天的地方），花曦说："这里是我们开始的地方。"然后她做了一件只有在主角面前才会做的事——解开扣子，坐在长椅上，看河面上的光。

这个Scene的功能从"关系公开"变为"回到起点，但已经不同了"——更私密，更有余韵。

#### 3.5.4 友情线扩充

v6友情线仅数句对话，严重单薄。v7应：
- 增加友情线的专属Scene（至少1个完整Scene）
- 友情线的核心不是"遗憾"，而是"另一种珍贵"——花曦选择友情线后，她依然成长，依然破茧，只是主角以朋友的身份见证
- 友情线终章应有独立的情感高潮，不是爱情线的"缩减版"

#### 3.5.5 Scene 5-3 终章——全线情感最高点

v7目标：情感峰值~9.5，超越4-4天台事件，成为全线真正的最高点。

关键设计：
- 终章不应只是"回顾+告别"，应有一个"新的开始"的承诺
- 糖纸回廊的设计保留，但增加一个"花曦主动做的新事情"——不是叠糖纸，而是画了一幅新画/做了一个新决定
- 爱情线终章的核心意象：从"碎掉的光"到"完整的光"——花曦不再画碎掉的夜景，她画了完整的星空
- 友情线终章的核心意象：糖纸不再需要叠起来记住——因为人还在，记忆就在

## 六、扣子象征线轨迹

### 6.1 v6轨迹问题

| 问题 | 说明 |
|------|------|
| 中段停滞 | 阶段1-4（S2-1到S3-2）均为"全扣"，4个Scene无可见变化 |
| 阶段7是"作弊" | 4-7穿连衣裙无扣子，不是"解开扣子"而是"换了一件衣服"，象征力度弱于主动解开 |
| 缺少"想解但不敢"的过渡 | 从"全扣"直接跳到"扣不紧"，缺少花曦内心挣扎的外化 |

### 6.2 v7十阶段轨迹

| 阶段 | Scene | 扣子状态 | 情感含义 | 独处？ | 变化类型 |
|:----:|-------|---------|---------|:------:|---------|
| 1 | 2-1 初见 | 全扣·紧·标准 | 在所有人面前扮演完美 | ❌ | 基准 |
| 2 | 2-2 河岸 | 全扣·手指碰了一下领口又放下 | 有一瞬间想松，但习惯压住了 | ✅ | 微动 |
| 3 | 2-5 文化祭后台 | 全扣·但肩膀弧度软了 | 卸下表演后的松弛，但扣子仍是防线 | ✅ | 间接 |
| 4 | 3-2 河岸 | 全扣·解了第一颗又扣回去 | 想放松但不敢——手比心诚实 | ✅ | **关键过渡** |
| 5 | 3-5 糖纸秘密 | 全扣·指尖停在第二颗扣子上 | 信任到了，但还没准备好 | ✅ | 临界 |
| 6 | 4-3 能量下降 | 全扣·但扣不紧 | 不是主动松开——是撑不住 | ✅ | 被动 |
| 7 | 4-5 陪伴 | **第一颗扣子松开** | 第一次在他面前主动松开——不是因为撑不住，是因为想松 | ✅ | **主动突破** |
| 8 | 4-7 画展 | 无扣子（便服·连衣裙） | 角色转换——不需要扣子了 | ✅ | 身份转换 |
| 9 | 5-2 告白 | **三颗全解** | 完全信任·不再伪装 | ✅ | 完整交付 |
| 10 | 5-3 终章 | 校服·三颗全解 | 毕业日·彻底的敞开——扣子不再是防线，因为不需要防线了 | ✅ | 终极自由 |

**v7关键改进**：
- **阶段4（解了又扣回去）**：这是"想解但不敢"的具象化——花曦的手比她的心更早做出选择，但理智把动作按了回去。这个细节比v6的"语气松弛"更有画面感。
- **阶段5（指尖停在扣子上）**：与3-5糖纸秘密同步——信任已经到了，但还需要一个契机才能迈出那一步。
- **阶段7（第一颗主动松开）**：这是全线最重要的扣子变化——不是"扣不紧"的被动，不是"换衣服"的取巧，是花曦在主角面前**选择**松开。这个动作应与4-5中的某个关键对话同步（如：花曦说出"在你面前我可以不撑"之后，手伸向领口）。
